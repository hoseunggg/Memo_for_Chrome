Privacy Policy for Memo for Chrome

Memo for Chrome is a Chrome extension that lets users create, edit, organize, and sync text memos with their own Google Drive.

Data We Handle

The extension may store user-created memo text, file and folder names, app settings, and sync state locally in Chrome storage. If Google Drive sync is enabled, memo files are synced to the user's own Google Drive account.

Google Drive Access

Memo for Chrome uses Google Drive API access only to read, create, update, move, and delete memo files for synchronization. The extension does not access files for advertising, analytics, or unrelated purposes.

Data Sharing

The developer does not sell, rent, or transfer user data to third parties. User memo data is not sent to any external server owned by the developer.

Authentication

Google OAuth is used only to allow the extension to sync memo files with the user's Google Drive. The developer does not collect Google account passwords.

Data Retention

Memo data remains in the user's Chrome storage and/or Google Drive until the user deletes it. Users can remove local data by uninstalling the extension or clearing extension storage, and can delete synced files directly from Google Drive.
